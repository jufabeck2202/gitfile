# gitfile [![Build Status]

> download single files from github 


## Install

```
$ npm install -g gitfile
```


## Usage
```bash
    gitfile <GitHubFileURL/GistFileURL>
    gitfile <filename> <githubURL>
```
Directly download files into the current folder.
